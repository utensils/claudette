Features

Everything you need, nothing you don't

Purpose-built for Claude Code workflows. Lightweight, fast, and delightful to use.

🤖

Parallel Agents

Run multiple Claude Code agents simultaneously, each in its own isolated workspace.

🌳

Git Worktrees

Native git worktree integration for branch-based parallel development without conflicts.

🌐

Remote Workspaces

Connect to remote machines via encrypted WebSocket with automatic mDNS discovery.

💻

Integrated Terminal

Multi-tab terminal with xterm.js, right alongside your agents and code diffs.

📝

Diff Viewer

Real-time git diff tracking with unified and side-by-side views plus syntax highlighting.

🎨

Theming

Beautiful dark theme with CSS custom properties. Make it yours with full customization.

⚡

Lightweight

Under 30MB binary, under 2s cold start. Native Rust performance without Electron bloat.

⚙️

Per-repo Settings

Custom instructions, setup scripts, and icons tailored to each of your repositories.

Get Started

Up and running in minutes

Three simple steps to supercharge your Claude Code workflow.

1

Add your repos

Connect local or remote git repositories. Set up custom instructions and scripts for each.

2

Spin up workspaces

Each workspace gets its own git worktree and dedicated Claude Code agent session.

3

Work in parallel

Multiple agents coding simultaneously, each in full isolation. Review diffs, merge, and ship.

Built With

Modern tech, native feel

🦀 Rust 📦 Tauri 2 ⚛️ React 📘 TypeScript 🗄️ SQLite 📟 xterm.js

Built with Tauri 2 for a truly native experience without the Electron overhead. Rust backend for speed, reliability, and a tiny footprint.
