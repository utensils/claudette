## Escape Key Priority

When multiple overlays are open, `Escape` dismisses them in this order:

1. Command Palette
2. Settings
3. Modal dialogs
4. Fuzzy Finder
5. Diff panel
6. Stop running agent

## Keyboard Hint Badges

When you hold `‚åò` (macOS) or `Ctrl` (Linux) without pressing another key, Claudette shows shortcut badges on relevant UI elements ‚Äî sidebar toggle, diff panel, terminal, and settings buttons. Release the modifier key to hide them.
