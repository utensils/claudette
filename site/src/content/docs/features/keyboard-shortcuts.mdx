Claudette supports keyboard shortcuts for fast navigation and common actions. Hold `‚åò` (macOS) or `Ctrl` (Linux) to reveal shortcut badges on UI elements. Tooltips and command-palette shortcut labels use your current **Settings ‚Üí Keyboard** bindings, so disabled or rebound shortcuts are reflected in the UI.

### Tabs & Navigation

The unified tab strip holds chat sessions, diff tabs, file tabs, and any other workspace tabs in one drag-reorderable row. The bracket shortcuts cycle the strip in order, with wrap-around.

| Shortcut | Action |
|----------|--------|
| `‚åò/Ctrl + T` | New tab ‚Äî chat session by default, or new file when a file is open |
| `‚åò/Ctrl + W` | Close the active tab |
| `‚åò/Ctrl + Shift + [` | Previous tab in the unified tab strip |
| `‚åò/Ctrl + Shift + ]` | Next tab in the unified tab strip |
| `‚åò/Ctrl + 1` ‚Äì `‚åò/Ctrl + 9` | Jump to project by index |
| `‚åò/Ctrl + Shift + N` | New workspace in the current project |

`‚åò/Ctrl + Shift + N` resolves which project to target in this order: the project-scoped view's pinned repo ‚Üí the active workspace's repo ‚Üí the first local repo. The new workspace is auto-named, the parent repo group is force-expanded in the sidebar, and the workspace is selected so you land in its chat view.

To switch between workspaces in the same project, click them in the sidebar or use the fuzzy finder (`‚åò/Ctrl + K`) ‚Äî the bracket shortcuts no longer specifically cycle workspaces, since the unified tab strip subsumed that intent.

### Terminal Panes

These fire only when a terminal pane has keyboard focus ‚Äî they shadow the
equivalent global shortcuts where applicable (e.g. `‚åòD` toggles the diff
panel when the chat is focused but splits the terminal on macOS when the
terminal is focused; Linux/Windows keep bare `Ctrl+D` for shell EOF and use
`Ctrl+Shift+D` instead).

| Shortcut | Action |
|----------|--------|
| `‚åòD` (macOS) / `Ctrl+Shift+D` (Linux) | Split pane side-by-side |
| `‚åò‚áßD` (macOS) / `Ctrl+Shift+Alt+D` (Linux) | Split pane stacked |
| `‚åòW` (macOS) / `Ctrl+Shift+W` (Linux) | Close the focused pane (or the tab when it's the last pane) |
| `‚åò‚å•‚Üê/‚Üí/‚Üë/‚Üì` (macOS) / `Ctrl+Alt+arrow` (Linux) | Move focus to the neighboring pane |
| `‚åòT` (macOS) / `Ctrl+Shift+T` (Linux) | New terminal tab |

| `‚åò/Ctrl + =` / `‚åò/Ctrl + -` | Global UI zoom; terminal intercepts the keys so they do not reach the shell |

## Escape Key Priority

When multiple overlays are open, `Escape` dismisses them in this order:

1. Command Palette (whether opened via `‚åò/Ctrl + Shift + P` for command mode or `‚åò/Ctrl + P` for Quick Open file mode)
2. Settings
3. Modal dialogs
4. Fuzzy Finder
5. Diff panel
6. Stop running agent

Inside Settings, `Escape` peels off one layer at a time: open dropdowns (font picker, default-terminal picker, native `<select>`), rebind capture, and inline editors (pinned prompts) are dismissed first. Only when no inner overlay is active does `Escape` exit Settings and return to the app ‚Äî `‚åò/Ctrl + ,` and the "Back to app" control remain available as direct exits.

Stopping an agent with `Escape` preserves queued chat messages and pauses automatic queue delivery until you choose what to send next.

## Keyboard Hint Badges

When you hold `‚åò` (macOS) or `Ctrl` (Linux) without pressing another key, Claudette shows shortcut badges on relevant UI elements ‚Äî sidebar toggle, diff panel, terminal, and settings buttons. Release the modifier key to hide them. Hover tooltips show the same current binding when the related shortcut is enabled.
